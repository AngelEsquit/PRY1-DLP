# Casos de uso de IDE (Legacy)

La UI antigua basada en tkinter fue retirada del proyecto.

Este documento se conserva solo como referencia histórica de los casos manuales que existían para esa interfaz.

Estado actual:

- El flujo activo del backend Python es por CLI: `python src/main.py`.
- La experiencia tipo IDE actual es la app desktop en `desktop-app/` (Tauri + React).

Para casos de uso vigentes de la app desktop, ejecutar:

```bash
cd desktop-app
npm run tauri dev
```
